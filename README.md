## About the Game
*Ledger and Lightyear* brings the depth of galactic strategy to the simplicity of a paper notebook. It is designed to be played anywhere you have a pen and a grid, removing the need for expensive miniatures or complex board setups.

### Key Features
* **The Brick-Grid System:** A unique mapping method that allows for consistent movement and exploration on standard notebook paper.
